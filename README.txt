การ setup โปรแกรม

- เมื่อเปิดโปรแกรมครั้งแรก จะมีการสร้างไฟล์ user.csv จากนั้นจะทำการอ่านค่าใน user.csv ซึ่งจะกำหนด username และ password ไว้ 2 ชุด จากนั้นจะนำมาเก็บไว้ในตัวแปรแกรม

- เมื่อมีการเปลี่ยน scene จะมีการส่งค่าตัวแปรไปยังหน้าถัดๆไปโดยใช้ method set เช่น user themes ticket

- ในหน้า setting จะมีการอ่านไฟล์ user.csv และ movie.csv ถ้าเป็นการเปิดครั้งแรกโปรแกรมจะสร้างไฟล์ขึ้นมาใหม่โดยกำหนดค่าเริ่มต้นไว้ทั้งสองไฟล์ สำหรับผู้ใช้ที่เป็น admin จะสามารถเข้าไปยัง menu movie setting , user setting ได้
 ซึ่งในหน้า movie setting จะมีปุ่มสำหรับแก้ไขและเพิ่มรอบของหนัง การใช้งานปุ่มทั้งสองปุ่มนี้ระบบจะไปเขียนและบันทึกข้อมูลลงในไฟล์ movie.csv ส่วนในหน้า user setting การกดปุ่ม set , delete จะเป็นการเขียนไฟล์ลงใน user.csv

-ในการเพิ่มโปรเตอร์หนัง หรือ ตัวอย่างหนัง จะบันทึกตำแหน่งของไฟล์รูปหรือตัวอย่างลงไปในไฟล์ movie.csv โดยจะแนบโปรเตอร์และตัวอย่างสำหรับทดสอบไปใน folder test 

-ในหน้าสำหรับจองที่นั่ง เมื่อเปิด scene จะทำการอ่านไฟล์ BookingData.csv แล้วนำข้อมูลไปเป็นเงื่อนไขในการแสดงรูปภาพสถานะของเก้าอี้ ถ้าเป็นการเปิดครั้งแรกจะสร้างไฟล์เปล่าขึ้นมา และหลังจากเลิกที่ทั้งเสร็จแล้วกดปุ่มคอนเฟิร์มและกดยืนยัน จะทำการเขียนที่นั่งที่จองแล้วลงไปในไฟล์ bookingData.csv
 ในกรณีที่เป็นการยกเลิกเก้าอี้ที่เคยจองไว้ ด้วย username เดียวกัน เมื่อกดเลือกเก้าอี้นั้น แล้วกดยืนยันยกเลิกการจอง ระบบจะเขียนข้อมูลใหม่ลงไปในไฟล์ BookingData.csv ทันทีโดยไม่ต้องกดคอนเฟิร์ม

-เมื่อก่อนยืนยันการจอง และเปิน scene ของ ticket ระบบจะ save ตั๋วทุกใบแล้วเก็บไว้ใน folder ticket และแสดงหน้าตั๋วใบล่าสุด และใบเสร็จการทำรายการ เมื่อกด save bill ใบเสร็จจะถูก save เป็นรูปเก็บไว้ใน folder bill
